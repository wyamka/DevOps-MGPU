Данный датасет был получен из открытого набора данных на платформе Kaggle [Diabetes Data Set](https://www.kaggle.com/datasets/mathchi/diabetes-data-set)

Для целей выполнения проектной задачи Варианта 25 в исходный набор данных были добавлены дополнительные поля FirstName и LastName. Эти поля предназначены для демонстрации работы ETL-контейнера (loader) по анонимизации персональных данных перед загрузкой в базу данных, что позволяет сохранить конфиденциальность информации пациентов при дальнейшей аналитической обработке.
